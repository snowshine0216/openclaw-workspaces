# Setup

## Required before first full run

- Create or provide a Notion integration token at `~/.config/notion/api_key`
- Share the target Notion parent page with the integration
- Verify GitHub CLI authentication if GitHub collection is enabled
- Verify Twitter/X collection path if Twitter is enabled
- Verify Reddit collection path if Reddit is enabled

## Runtime expectations

The skill is designed around `agent-reach` guidance:
- use upstream tools directly after installation
- keep research collection compatible with `xreach`, `gh`, `curl`, and `mcporter`
