# Source Policy

## Supported source types

- Web/news
- GitHub
- arXiv
- Twitter/X
- Reddit

## Source rules

- Prefer high-signal items over volume
- Include source URLs for every reported item
- Include direct PDF URLs for `arXiv`
- Treat Reddit as secondary signal, not the main feed
- Use Twitter/X for frontier signals, releases, and expert commentary

## Filtering intent

Prioritize:
- model releases
- research breakthroughs
- meaningful GitHub tools
- notable papers worth reading
- repeated themes across sources
