# Summary Format

## Daily digest

Include:
- executive summary
- key highlights
- suggested actions
- learnings
- source links

## Weekly digest

Include:
- week in one paragraph
- biggest themes
- top repos to try
- top papers to read
- top social/community signals
- suggested actions for next week
- key learnings
- source links

## Shareability

- Weekly digest should be polished and readable out of context
- Daily digest should stay concise but clear
- Every major claim should be traceable to a source URL
